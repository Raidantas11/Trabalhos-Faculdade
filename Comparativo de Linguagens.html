Phyton Python é uma linguagem de programação de alto nível, interpretada de script, imperada, orientada a objetos, funcional, de tipagem dinâmica e forte. Pelo pouco contato que tive com linguagens aqui expostas, talvez o Python tenha sido uma das que mais me agradaram pela praticidade que oferece, rapidez, mas ainda me causou estranhesa justamente por pular etapas daquela "receita"
C C é uma linguagem de programação compliada de propósito geral, estruturada, imperativa, procedural, padronizada pela Organização Internacional para Padronização, criada em 1972 por Dennis Ritchie na empresa AT&T Bell Labs para desenvolvimento do sistema operacional Unix. Apesar da minha péssima experiencia em aprender C, talvez tenha sido a que eu mais tenha gostado. Achei até certo ponto intuitivo, seguindo passo a passo, talvez porque achei parecido com o Portugol
Javascript JavaScript é uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma. Juntamente com HTML e CSS, o JavaScript é uma das três principais tecnologias da World Wide Web. Achei aqui um pouco mais chatinho, talvez por pouco contato. Mas o JS traz toda a interatividade e da toda vida aos sites, então muito me agrada e me traz curiosidade.
Java Java é uma linguagem multiplataforma, orientada a objetos e centrada em rede que pode ser usada como uma plataforma em si. É uma linguagem de programação rápida, segura e confiável para codificar tudo, desde aplicações móveis e software empresarial até aplicações de big data e tecnologias do servidor. C# C# é uma linguagem de programação, multiparadigma, de tipagem forte, desenvolvida pela Microsoft como parte da plataforma .NET. A sua sintaxe orientada a objetos foi baseada no C++ mas inclui muitas influências de outras linguagens de programação, como Object Pascal e, principalmente, Java. Ainda não tive a oportunidade de mexer com a linguagem. Apenas conheço por alguns comentários e por procurar saber quais linguagens eram usadas para fazer alguns jogos. Mas pelo que li, sua versatilidade pode ser o mais encantador da linguagem.
PHP O PHP (um acrônimo recursivo para PHP: Hypertext Preprocessor) é uma linguagem de script open source de uso geral, muito utilizada, e especialmente adequada para o desenvolvimento web e que pode ser embutida dentro do HTML. Outra linguagem que ainda não tive contato, porém ao que parece, se assemelha ao Python e JS na sua forma mais prática de escrever um código.
C++ C++ é uma linguagem de programação de alto nível e de propósito geral. Ela foi desenvolvida como uma extensão da linguagem C, incorporando recursos adicionais, como suporte à programação orientada a objetos. Vamos te mostrar porque você precisa dominá-la para uma carreira de sucesso no universo tecnológico. O C++ me parece interessante pela vasta utilização em vários tipos de aplicações, e em jogos. Como parece que tem um uso de memória otimizado, deve permitir uma maior eficiência na hora de entregar um produto para o cliente.
